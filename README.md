# Витакод

Витакод — мобильное PWA-приложение с концепцией цифрового двойника:
- питание, БЖУ, витамины, минералы, аминокислоты;
- учет воды и клетчатки;
- трекер БАДов (включая мультивитаминные капсулы);
- тренировочный модуль (волновая периодизация, КПШ, тоннаж, прогноз ПМ);
- оценка согласованности данных и поиск аномалий режима.

## База данных

Используется локальная расширяемая база:
- `food-db.js` — продукты и блюда;
- `supplement-db.js` — БАДы.

## Важно

Рекомендации в приложении не заменяют медицинскую консультацию.
